If you want to get involved in making ETS Framework better, please feel free to submit a pull request with any additions or changes you think would be suitable.

Alternatively, if you're not a coder, or don't want to submit a pull request, feel free to [submit an issue](https://github.com/ets-tx/ets-framework/issues).

## Contributing

If you decide to contribute to this project, please edit the `app-framework.css`. Once merged, minification will be handled separately.

If you do add contributions, please add verbose commenting so the rationale for the changes/additions is clear.

## Guidelines

- Follow existing code style (logical property grouping by rendering pipeline)
- Ensure accessibility (WCAG AA minimum)
- Test on mobile devices
- Keep file size under 20KB
- Document any new CSS variables

Thanks for wanting to contribute to this project!
