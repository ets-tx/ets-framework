Paragraphs

A paragraph (from the Greek paragraphos, “to write beside” or “written beside”) is a self-contained unit of a discourse in writing dealing with a particular point or idea. A paragraph consists of one or more sentences. Though not required by the syntax of any language, paragraphs are usually an expected part of formal writing, used to organize longer prose.

[Top]

Blockquotes

A block quotation (also known as a long quotation or extract) is a quotation in a written document, that is set off from the main text as a paragraph, or block of text.

It is typically distinguished visually using indentation and a different typeface or smaller size quotation. It may or may not include a citation, usually placed at the bottom.

.visually-hidden

The .visually-hidden class hides content visually but keeps it accessible to screen readers. Used for skip navigation links at the top of this page.

Try it: Press the Tab key when this page loads to see the skip links appear.

<a href="#main" class="visually-hidden">Skip to main content</a>

Automatic dark mode

ETS Framework automatically adapts to your system's color scheme preference using prefers-color-scheme.

Try it: Change your operating system's theme from light to dark mode to see the framework adapt instantly.

No classes or JavaScript required—it's built into the CSS and respects user preferences.

[Top]

Keyboard navigation

All interactive elements in ETS Framework are fully keyboard accessible following WCAG 2.1 Level A requirements.

Navigation keys

Tab
Move forward through interactive elements
Shift + Tab
Move backward through interactive elements
Enter
Activate links, buttons, and submit forms
Space
Activate buttons and toggle checkboxes/details
Arrow Keys
Navigate within radio button groups and select menus
Esc
Close dialogs and modals

Focus indicators

All focusable elements display a clear 3px outline in the accent color with 2px offset when focused via keyboard.

Try it: Press Tab repeatedly on this page to see focus indicators on all interactive elements.
