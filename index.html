Style Guide

Overview

ETS-TX v26.0α is a modern, accessible CSS framework designed for admin interfaces and LMS applications. Built with a focus on performance, accessibility, and developer experience.

Key Features

  • ≤25KB CSS — Lightweight, no dependencies
  • WCAG 2.1 AA+ — Fully accessible out of the box
  • 3 Themes — Light, Dark, and High Contrast
  • Mobile-First — Responsive 8→12→16 column grid
  • System Fonts — No web fonts, faster load times
  • Semantic HTML — Classless where possible

Browser Support

95%+ global browser coverage:

  • Chrome 90+, Edge 90+
  • Firefox 88+
  • Safari 14+
  • iOS Safari 14+
  • Samsung Internet 14+

Installation

Add the CSS and JS files to your project:

HTML Setup

<!DOCTYPE html>
<html lang="en" class="no-js">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">

  <!-- Critical inline JS -->
  <script>(function(d,c){c.remove('no-js');c.add('js');var t=localStorage.getItem('ets-theme');if(t)d.dataset.theme=t})(document.documentElement,document.documentElement.classList);</script>

  <link rel="stylesheet" href="app-framework.css">
</head>
<body>
  <!-- Theme anchors -->
  <span id="light"></span>
  <span id="dark"></span>
  <span id="high-contrast"></span>

  <!-- Your content -->

  <script src="app-framework.js"></script>
</body>
</html>

Accessibility

Built-in accessibility features for WCAG 2.1 AA compliance.

Focus Styles

All interactive elements have visible focus indicators. Tab through this page to see them in action.

Screen Reader Announcements

Use ETS.announce() to announce changes to screen readers:

ETS.announce('Form submitted successfully'); ETS.announce('Error: Please check your input', 'assertive');

Reduced Motion

Animations are automatically disabled when prefers-reduced-motion: reduce is set in the user's system preferences.

High Contrast Mode

The high contrast theme provides maximum contrast ratios for users with low vision. All colors become pure black/white with yellow accents, and shadows are replaced with solid borders.
